Detailed Description

balde request context

This struct stores everything related to the request context. It stays loaded in memory during all the request life time, and is initialized automatically by the application main loop.

Field Documentation

GHashTable* args

A GLib hash table that stores the query string parameters.

Do not touch it manually, use the balde_request_get_arg() function instead.

GHashTable* form

A GLib hash table that stores the parsed form data.

Do not touch it manually, use the balde_request_get_form() function instead.

GHashTable* headers

A GLib hash table that stores the request headers.

Do not touch it manually, use the balde_request_get_header() function instead.

balde_http_method_t method

Request HTTP method.

Can be used to verify which method was used by the client doing the current request.

const gchar* stream

Raw request body, if provided by client.

You can use it, but not change its value.

GHashTable* view_args

A GLib hash table that stores the arguments parsed from the request path.

Do not touch it manually, use the balde_request_get_view_arg() function instead.
